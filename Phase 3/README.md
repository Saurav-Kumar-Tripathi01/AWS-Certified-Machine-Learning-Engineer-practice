# Phase 3: AWS Foundations for Developers

## Phase Objective
Understand how cloud infrastructure supports applications and ML systems.

You are not memorizing services — you are learning **how systems connect**.

---

## Core AWS Services

- IAM (roles and policies)
- S3 (data and model storage)
- EC2 (compute)
- CloudWatch
- AWS CLI and boto3

---

## Mini Projects

### Project 3.1: Python App Fetching Data from S3

**Objective**  
Learn how applications interact with cloud storage securely.

**Expectation**
- Upload data to S3
- Fetch it using Python
- Use IAM roles safely

**Success Criteria**
You understand secure data access in AWS.

---

### Project 3.2: EC2‑Hosted REST API

**Objective**  
Deploy a Python application on the cloud.

**Expectation**
- Launch EC2
- Run API server
- Expose it publicly

**Success Criteria**
You understand compute, networking, and deployment basics.

---

## Milestone #3

You can:
- Use IAM correctly
- Run Python services on AWS
- Explain how AWS components fit together
