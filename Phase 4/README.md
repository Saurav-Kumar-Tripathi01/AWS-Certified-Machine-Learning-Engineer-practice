## Capstone Project

### End‑to‑End ML Pipeline on AWS

**Objective**  
Design and implement a full ML pipeline that mirrors real production environments.

**Pipeline**
Data → S3 → SageMaker Training → Endpoint → Monitoring

**Expectation**
- Train model using SageMaker
- Deploy model as endpoint
- Invoke predictions via Python
- Monitor performance

**Success Criteria**
You can:
- Explain the system end‑to‑end
- Debug failures
- Discuss cost and scalability
