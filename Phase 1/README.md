# Phase 1: Core Developer Skills

## Phase Objective
Become a confident Python developer with strong engineering fundamentals.  
This is the **most important phase** of the entire roadmap.

If this phase is weak, ML and AWS will collapse later.

---

## What You Must Learn

### Python Foundations
- Variables, loops, and functions
- Lists, dictionaries, tuples
- Basic OOP (classes and objects)
- File handling
- Exception handling

### Developer Fundamentals
- Git & GitHub
- Virtual environments
- Reading documentation
- Writing clean, readable code

---

## Mini Projects

### Project 1.1: CLI To‑Do App

**Objective**  
Learn how real programs are structured and executed.

**Expectation**
- Handle user input
- Store tasks in memory or file
- Gracefully handle errors

**Success Criteria**
You can write scripts without fear and fix basic bugs yourself.

---

### Project 1.2: CSV Data Reader & Summary Tool

**Objective**  
Understand how developers work with files and structured data.

**Expectation**
- Read CSV files
- Compute summaries (rows, columns, basic stats)
- Handle invalid or missing data

**Success Criteria**
You can work confidently with data files.

---

### Project 1.3: Simple REST API (FastAPI / Flask)

**Objective**  
Understand how backend services expose functionality.

**Expectation**
- Create endpoints
- Handle requests and responses
- Run a local server

**Success Criteria**
You understand how APIs work at a basic level.

---

## Milestone #1 (Non‑Negotiable)

You should be able to:
- Write Python scripts confidently
- Debug simple issues independently
- Push projects to GitHub correctly
