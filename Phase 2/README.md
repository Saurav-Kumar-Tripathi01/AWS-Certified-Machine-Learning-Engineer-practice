# Phase 2: Data Handling and ML Basics

## Phase Objective
Step into machine learning by learning how data is prepared and how basic ML models work conceptually.

No advanced math.  
Understanding > equations.

---

## What You Must Learn

### Data Skills
- NumPy
- Pandas
- Data cleaning
- Basic feature engineering

### ML Fundamentals
- Supervised vs unsupervised learning
- Regression vs classification
- Train/test split
- Evaluation metrics

---

## Mini Projects

### Project 2.1: House Price Prediction

**Objective**  
Predict a numerical value using regression.

**Expectation**
- Clean dataset
- Train a regression model
- Evaluate using appropriate metrics

**Success Criteria**
You can explain *why* the model works conceptually.

---

### Project 2.2: Spam Email Classifier

**Objective**  
Classify emails into spam vs not‑spam.

**Expectation**
- Text preprocessing
- Train a classifier
- Understand false positives vs false negatives

**Success Criteria**
You understand classification tradeoffs.

---

### Project 2.3: Customer Churn Prediction

**Objective**  
Predict whether a customer will leave.

**Expectation**
- Handle categorical features
- Train ML model
- Interpret results

**Success Criteria**
You can connect ML outcomes to real business decisions.

---

## Milestone #2

You can:
- Load and clean data
